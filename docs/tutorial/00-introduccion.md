# Infusio — Guía de uso

**Infusio** es una tienda artesanal argentina de cafés de especialidad, tés, yerba mate e infusiones. Esta aplicación corresponde al rol del comprador dentro de un marketplace de tres apps (Seller, Buyer y Shipping).

**Deploy:** [https://proyecto-c-buyer-infusio.vercel.app](https://proyecto-c-buyer-infusio.vercel.app)
